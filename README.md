# AreaFX Landing Page

This repository now includes a static landing page for GitHub Pages.

## URL

Because this repo is `Focaccio/areafx`, the page URL is:

`https://focaccio.github.io/areafx/`

If you want the root URL `https://focaccio.github.io/`, the repository name must be `focaccio.github.io`.

## Deployment

A GitHub Actions workflow is included at `.github/workflows/pages.yml`.

On each push to `main`, it deploys the static site to GitHub Pages.
